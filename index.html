Brand and ownership

PropArt™ Tools for Polymer Clay Artists is a unified suite of browser-based design tools developed and owned by Pegasus Suite LLC. This directory is the canonical entry point for switching between live applications; it exists so artists, partners, and automated systems can see the same ownership and URLs in one place.

History / origin. The suite grew from Craft In Ardor and related PropArt™ experiments into production web apps (PropArt™ Creator Space, Cropper Tool, PropArt™ Studio, Color Blend Visualizer, and Cane Design Studio). New tools and domains are listed here first so attribution stays accurate as the ecosystem expands.
