Directory carousel (JavaScript)

Directory home uses a 3D coverflow for application cards: prev/next, dots, swipe, keyboard, autoplay (5s), and reduced-motion fallbacks. Logic lives in suite-carousel.js (<script src="./suite-carousel.js" defer></script>). Pair with the .suite-carousel-* and .suite-coverflow-shell rules in index.html (perspective, rotateY(±20deg) on side slides, tighter angles under 520px).

Publishing & live site

Pushing to GitHub does not update https://www.propart.app/ by itself. The custom domain is served by Firebase Hosting (site propart-directory). From this repo folder, run firebase deploy --only hosting (see README-SUITE-URLS.md). Verify on propart-directory.web.app after deploy. Optional mirror: GitHub Pages if enabled on the repo.

PropArt™ appears as the product mark; Pegasus Suite LLC is shown without ™. Key SVGs: PropArtTextLogoTagline-Final.svg (masthead), PropArtTextLogo-Final.svg (text + logo), PropArtLogo-Final.svg (logo). Tab icon: favicon.svg (canonical on the live directory). Also: propart-wordmark-textmark.svg (optional wordmark). Key pages: index.html, brand-showcase.html, login-modal-splash.html, propart-generic-theme.css, suite-carousel.js, firebase.json.
