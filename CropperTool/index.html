Cutout workflow

A slice here is best read as one object cutout: grouped regions that actually touch in the mask become one outline. Use Objects on plain backdrop (or Whole objects ON) when a few solids sit on off-white—not for tight grids where every cane slice must separate. Pro: auto sheet palette (25 swatches max) — tap a swatch to set background. Others: use the eyedropper beside tolerance. Browser heuristics only — not OCR.

Best for earrings, studs, sculptures on paper: avoids breaking one piece into many ovals/rects. Turn off + use Busy for dense cane grids where blobs must subdivide.

Welcome to PropArt™ Cropper Tool

Start in five steps

  1. Add an image: use Upload or Open project, or click the dashed drop frame to browse.
  2. Choose a crop shape: Rect for tiles and strips; Oval for round bits.
  3. Draw each crop: click-drag on your photo. Resize by drawing again — remove extras with Adjust slices (hover · left deletes · right for menu).
  4. Label & export: name your project in Profile → Project details; describe slices in exported JSON/PDF — then Download all or Save JSON.
  5. (Optional) Flat backgrounds: expand Background detection & auto-crop, Pick BG, tune Tolerance, then Auto-crop (large images yield occasionally so the tab stays responsive).
  6. Craft cloud & pattern library: Sign in · Save online / vault · duplicate a listing or restore a removed save. Toolbar More tools → book opens Pattern library (export/import packs, tags, rename, insert at offset / centre / pointer).

Toolbar layout

  • One row — Files, Draw, View, Help, Export, and Edit; scroll sideways if your window is narrow.
  • Expandable sections — advanced auto-crop tools and batch/AI/export sit in closed panels until you open them.

Browser DevTools “Issues” (not Cropper bugs)

If Chrome shows a number like “3 Issues” by the toolbar, expand the Issues drawer or tab beside Console. Those listings are warnings from the browser (security, cookies, form fields). PropArt logs that start with [PropArt] in the console are ours.

Tips

  • Slices toolbar: 1–3 shapes · 4 All slices grid · 5 Adjust slices (Move). Move: arrows nudge · optional Snap (grid · sheet · neighbors) · hover · left deletes · right menu · drag · Ctrl+D duplicate · slice recipe Ctrl+Shift+E. Gallery shows collapsible slice notes.
  • Zoom shortcuts: Ctrl++ / Ctrl+− / Ctrl0 (⌘ on Mac).
  • Pan: hold middle mouse button or Ctrl and drag.
  • Sheet palette (Pro) — histogram swatches from the photo appear under the main tool row automatically. Everyone can use the palette button (colors from slices) and the BG eyedropper.
